# DASH Over 50+ Health Management App

Complete cardiovascular health management application for adults over 50, based on the DASH (Dietary Approaches to Stop Hypertension) diet methodology.

## Features

### Dashboard

-   Real-time health statistics
-   Daily checklist for health goals
-   Quick action buttons
-   Integrated AI Coach widget

### Meal Planning

-   30-day DASH diet meal plan
-   120+ complete recipes
-   Detailed nutritional information
-   Meal logging system

### Exercise Programs

-   12 workout programs (Beginner, Intermediate, Advanced)
-   Built-in workout timer
-   Step-by-step instructions
-   Progress tracking

### Blood Pressure Tracking

-   Easy BP reading input
-   Interactive charts and graphs
-   Automatic categorization
-   Historical data and trends
-   CSV export functionality

### Medication Management

-   Complete medication tracking
-   Smart reminder system with notifications
-   Customizable schedules (1x-4x daily)
-   Weekly scheduling
-   Medication history

### Progress Tracking

-   Journey overview with statistics
-   Health charts (BP trends)
-   Achievement system
-   Activity counters

### AI Health Coach

-   Interactive chat interface
-   Quick action buttons
-   Context-aware health advice
-   DASH diet guidance

### Education

-   5 Pillars of DASH methodology
-   Comprehensive health information
-   Nutrition tips
-   Best practices for 50+

### Settings

-   Profile management
-   Notification preferences
-   Theme switcher (light/dark)
-   Data export/import
-   Privacy Policy and Terms of Service

## User Guide

### Getting Started

1.  Onboarding: Complete the 4-step setup process
2.  Dashboard: View your health overview
3.  Set Goals: Configure your health targets
4.  Track Daily: Log meals, BP, medications, and exercise
5.  Monitor Progress: Check your achievements and trends

### Key Features Usage

#### Medication Reminders

1.  Go to Medications section
2.  Click "Add Medication"
3.  Enter medication details
4.  Set reminder schedule
5.  Enable notifications in browser

#### Blood Pressure Tracking

1.  Go to Blood Pressure section
2.  Enter systolic and diastolic readings
3.  View automatic categorization
4.  Check trends in interactive chart
5.  Export data for doctor visits

#### Meal Planning

1.  Go to Meals section
2.  Browse 30-day meal plan
3.  Click any meal for details
4.  View ingredients and instructions
5.  Log meals you have eaten
